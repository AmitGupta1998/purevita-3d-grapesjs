
# PureVita 3D GrapesJS

A powerful 3D website design tool built with GrapesJS, enabling users to create stunning three-dimensional web experiences without coding.

## Features

- **3D Design Canvas**: Intuitive visual editor for 3D website creation
- **GrapesJS Integration**: Leverages the powerful GrapesJS framework
- **Drag-and-Drop Interface**: Easy component placement and manipulation
- **Responsive Design**: Create mobile-friendly 3D websites
- **Real-time Preview**: See changes instantly

## Installation

```bash
npm install
npm start
```

## Usage

1. Open the application in your browser
2. Drag components onto the canvas
3. Customize 3D properties in the settings panel
4. Export your website
